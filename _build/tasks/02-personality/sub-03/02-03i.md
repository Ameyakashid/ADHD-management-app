## Changes Made

### 1. Redundant Exception Tuple Fix

**File:** `state_response_integration.py:137`

Changed `except (ValueError, Exception) as exc:` to `except Exception as exc:`. `ValueError` is a subclass of `Exception`, making the tuple equivalent to catching `Exception` alone. The fallback's purpose is to never let state detection crash the bot, so catching all exceptions is the correct intent.
