# Task 08 — Dashboard

## What

An always-on display on the Fire Tablet showing current state, upcoming tasks, buffer levels, and recent activity. A passive awareness surface — the user glances at it without needing to interact.

## Depends On

Task 03 (tasks), Task 05 (schedule), Task 06 (buffers).

## Produces

- A display interface optimized for Fire Tablet screen
- Current cognitive state indicator
- Upcoming tasks and schedule
- Buffer level summaries
- Recent activity feed
- Auto-refresh without user interaction

## User Constraints

- Fire Tablet is the dedicated always-on display hardware.
- Must work alongside Telegram (not replace it).
- Passive — glanceable, not interactive.

## Supervisor Focus

- What display technology works on Fire Tablet (web app, Telegram channel, other)
- How to keep the display updated in real-time or near-real-time
- How to make it glanceable (information hierarchy, visual design for ADHD)
- Power and network considerations for always-on display
